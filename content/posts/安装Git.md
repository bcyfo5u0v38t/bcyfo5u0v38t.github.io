## 1.安装Git

`paru -S git`

## 2.配置

为了使用Git 至少需要设置一个名字和电子邮件  
`git config --global user.name "名字"`  
`git config --global user.email "电子邮件"`

## 3.安装GitHub-Desktop

这是一个用于GitHub的桌面版本  
`paru -S github-desktop`
