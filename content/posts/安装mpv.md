## 1.安装mpv

`paru -S mpv`

## 2.配置

复制默认设置的示例配置文件  
`cp -r /usr/share/doc/mpv/ ~/.config/`  
配置  
`nvim ~/.config/mpv/mpv.conf`  
一些通用配置

```
sub-auto=fuzzy    #字幕模糊搜索
sub-bold=yes    #加粗字幕增加可读性
```

高品质配置

```
profile=gpu-hq    #在视频输出时加载高质量的OpenGL
scale=ewa_lanczossharp    #gpu-hq默认采用spline36 缩放过滤器并可以提供中等的速度和质量
cscale=ewa_lanczossharp    #为了使视频输出质量达到最优 如果你的硬件允许的话 你应该选择ewa_lanczossharp
video-sync=display-resample    #在音频和视频不同步时 重新采样音频 而不是丢失视频帧
interpolation    #使用插值模式来提高视频质量
tscale=oversample    #使用oversample插值模式来提高视频质量
```

## 3.使用Anime4K

安装  
`paru -S anime4k`  
把着色器软连接到那里  
`sudo ln -s /usr/share/anime4k /home/bcyfo5u0v38t/.config/mpv/shaders`

`nvim ~/.config/mpv/input.conf`
